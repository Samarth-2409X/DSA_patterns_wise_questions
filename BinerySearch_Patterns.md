# 🔍 Binary Search Patterns 

This repository organizes Binary Search problems into **core patterns** to help in quick recognition during interviews.

---

## ✅ Pattern 1: Classic Binary Search

**Idea:** Search for a target in a sorted array.

**Problems:**

1. Binary Search
2. Guess Number Higher Or Lower

---

## ✅ Pattern 2: Lower Bound / Upper Bound

**Idea:** Find position (first ≥ target or first > target).

**Problems:**

1. Search Insert Position
2. Time Based Key Value Store

---

## ✅ Pattern 3: Binary Search on Answer ⭐

**Idea:** Search in the answer space using a feasibility check.

**Problems:**

1. Koko Eating Bananas
2. Capacity to Ship Packages Within D Days
3. Split Array Largest Sum

---

## ✅ Pattern 4: Rotated Sorted Array

**Idea:** One half is always sorted — decide search space accordingly.

**Problems:**

1. Search In Rotated Sorted Array
2. Search In Rotated Sorted Array II

---

## ✅ Pattern 5: Find Minimum / Peak

**Idea:** Use structure of array (not target) to decide direction.

**Problems:**

1. Find Minimum In Rotated Sorted Array
2. Find in Mountain Array

---

## ✅ Pattern 6: Binary Search on Matrix

**Idea:** Treat matrix as sorted (1D or row-wise).

**Problems:**

1. Search a 2D Matrix

---

## ✅ Pattern 7: Binary Search on Numbers (Math)

**Idea:** Search in numeric range using conditions.

**Problems:**

1. Sqrt(x)
